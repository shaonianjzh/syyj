# 森源艺佳项目
> 一个家具展示网站，前端主要展示一些家具的详细情况，后端管理端进行一些数据的管理。项目简单,是学完ssm后练手的第一个项目，主要实现了基本的增删改查功能。
## 技术栈

### 后端：
- Spring
- Spring MVC
- Spring JdbcTemplate
- JSP
- Tomcat 9.0.56
### 前端
- HTML + CSS
- Layui
- JQuery
- Bootstrap
