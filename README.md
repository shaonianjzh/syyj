森源艺佳项目
> 项目简单,是学完ssm后练手的第一个项目，主要实现了基本的增删改查功能。
> 持久层（Dao层）本来想使用mybatis框架，但是环境一直配不好，所以使用的是JDBCtemplate，有时间替换成mybatis框架。
## 用到的技术
#### 后端：
- spring+spring mvc+spring jdbctemplate
- 后端页面：layui 框架
